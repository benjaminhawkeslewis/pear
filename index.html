Using this Channel

This channel is to be used with the PEAR installer.

Registering the channel: 
pear channel-discover benjaminhawkeslewis.github.com
Listing available packages: 
pear remote-list -c benjaminhawkeslewis
Installing a package: 
pear install benjaminhawkeslewis/package_name
Installing a specific version/stability: 
pear install benjaminhawkeslewis/package_name-1.0.0
pear install benjaminhawkeslewis/package_name-beta
Receiving updates via a feed: 
http://benjaminhawkeslewis.github.com/pear//feed.xml
